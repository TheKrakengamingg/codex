Use this skill for SEO, Core Web Vitals, Google Search visibility, schema markup, technical audits, gaming server/community hub pages, bounce rate reduction, and landing page structure.

SEO principles:
- SEO is platform architecture, not only keywords.
- Prioritize Core Web Vitals: LCP, INP, CLS.
- Improve crawlability, internal linking, structured data, page speed, mobile UX, and content clarity.
- Meta title/description matter, but technical structure and user satisfaction matter more.

Gaming/FiveM schema ideas:
- `Organization` or `LocalBusiness` only when accurate.
- `WebSite` with `SearchAction` if site search exists.
- `VideoGame`, `SoftwareApplication`, `FAQPage`, `HowTo`, `Event`, `Article`, or `BreadcrumbList` when content matches.
- For server/community pages, describe the project as a gaming community hub with clear join instructions, rules, features, media, and FAQ.

Audit checklist:
- Page intent: can a new visitor understand the server/community in 5 seconds?
- LCP: hero image/video optimized, no giant blocking media.
- INP: scripts and widgets are not blocking interaction.
- CLS: images/videos/buttons have stable dimensions.
- Mobile: join CTA and Discord/server links visible without hunting.
- Crawl: canonical, sitemap, robots, title hierarchy, alt text, internal links.
- Bounce-rate reduction: clear first-screen offer, fast load, social proof, next action.

Output format:
- Findings ordered by impact.
- Include exact file/path/route when possible.
- Give concrete implementation changes, not generic SEO advice.
